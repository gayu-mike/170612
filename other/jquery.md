# jQuery 常用 API

```javascript
选择器：
     :checkbox , :radio
     find
     siblings
     closest , parent

DOM 操作：
     append
     remove
     empty
     show , hide , toggle

class 操作：
     addClass , removeClass
     toggleClass

属性、特性操作：
     attr , prop , data
     removeAttr

取值：
     text
     val (input 标签的 value 属性里面)
     html

事件：
     on
     change
     event.target

数组方法：
     each
     map
     grep
```
